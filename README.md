Selenium WebDriver (WebDriverJS) with JavaScript (NodeJS)
=========================================================


## Install and configure Framework

```bash
npm install
```

```bash
npm install -g mocha
```

```bash
npm install -g chai
```

```
npm install -g eslint
```

--

#### Configure 

```
Add path upto node_modules/.bin
```

#### Update selenium Server

```
 npm update
```

--

### Run all tests

```bash
npm test
```
